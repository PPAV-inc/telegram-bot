# PPAV Telegram Bot

## Install
```js
$ yarn
```

## Seed
```js
$ yarn run seed-setup
```  
edit `seed.json` in root folder  
```js
$ yarn run seed
```  


## Run
> Before run server, remember to create bot.config.js in **`/env`**  

`Development`
```js
$ yarn run dev
```  

`Production`
```js
$ yarn run start
```
